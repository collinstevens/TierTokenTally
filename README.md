Displays tier token distribution for your raid group via a minimap button and slash commands.

## Commands

- `/tiertokentally` — Print color-coded token counts to your chat frame
- `/tiertokentally report` — Post the token tally to raid chat

## Install

- [Wago](https://addons.wago.io/addons/tier-token-tally)
- [CurseForge](https://legacy.curseforge.com/wow/addons/tier-token-tally)
